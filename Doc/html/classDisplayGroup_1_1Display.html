Detailed Description

Single 7-segment display.

This class holds the representation of any digits within the single 7-segment display. The class shifts out the data pin and manages the clock pin to actually display the digits. This class behave like an observer in the "Observer" GoF Design Pattern: it observes the DisplayGroup to which belong and when its Display::update method gets called the actual value to display is converted in binary form and shifted out to the data output pin.

The shifting pattern follows the requirements for 74HC595 and 74HC164 shift register, which can be found in the datasheet.
