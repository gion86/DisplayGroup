Detailed Description

Manager for display groups. Add, remove, print and update all the displays at once.

Version
1.1

This class provides an easy way to manage many 7-segments display in an Arduino application. Each display is connected to a shift register, which holds the data until the next shift is carried out (see http://code.google.com/p/display-group/ for documentation). This class holds an array of groups, which in turn is an array of display, and update the whole array when requested. The update executes in reverse order, to account for the serial hardware link between the shift registers. The update can be done with latch pin low, and after the shift the latch pin is taken high, or with high output enable and transition to low on update. This is configurable through outputEnableState parameter on the constructor. LOW works with typical 74HC595 shift register.

This class uses the STL library for Arduino, which can be found at the-standard-template-library-stl-for-avr-with-c-streams
